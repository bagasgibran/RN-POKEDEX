## Running

#### Clone & install

* Clone this repo `git clone https://github.com/bagasgibran/RN-POKEDEX.git`
* `cd RN-POKEDEX`
* Run `npm install`
* Run `expo start`

#### Mobile Device
* Download Expo app
* Scan QR code from mobile or Open explore tab, Press search and Enter url shown in terminal.
